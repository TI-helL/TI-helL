# switch vs if

## Code Example

### switch

```md
switch(n) {
    case 1:
        ...
        break;
    
    case 2:
        ...
        break;
    
    ...
}
```

### if

```md
if (조건) {
    ...
}
else if (조건) {
    ...
}
else {
    ...
}
```

## feature comparison

* if문의 경우 명시한 모든 조건에 대해 검사한다.
* switch문의 경우 입력된 값에 해당되는 조건에 대해서만 코드를 수행한다.
